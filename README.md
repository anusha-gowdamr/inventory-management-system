# Inventory Management System

A web-based inventory management system developed using PHP, MySQL and Bootstrap.

## Features
- Product Management
- Category Management
- Supplier Management
- Sales Tracking
- Dashboard
- Search Functionality
- Login System
- Low Stock Alerts

## Technologies Used
- PHP
- MySQL
- Bootstrap
- HTML
- CSS
- JavaScript

## Installation
1. Download or clone the project
2. Move project folder to htdocs
3. Start Apache and MySQL in XAMPP
4. Import SQL database in phpMyAdmin
5. Open browser and run:
   http://localhost/project-folder

## Modules
- Products
- Categories
- Suppliers
- Sales
